# 💻 Meu Portfólio

Site pessoal para apresentar meus projetos como desenvolvedor, com um visual futurista neon e uma chuva de código animada ao fundo (estilo Matrix).

🔗 **[Ver site ao vivo](https://dioguinholf.github.io/MEU-PORTIFOLIO/)**

## ✨ Sobre

Portfólio construído do zero em **HTML, CSS e JavaScript puro** (sem frameworks), com foco em uma identidade visual forte e experiência interativa: fundo animado em canvas, menu responsivo com hambúrguer para mobile, e animações de entrada ao rolar a página.

## 🚀 Tecnologias

- **HTML5** — estrutura semântica
- **CSS3** — variáveis customizadas, Grid, Flexbox, media queries
- **JavaScript (vanilla)** — Canvas API, IntersectionObserver, manipulação de DOM
- **Google Fonts** — Rajdhani + Roboto
- **Font Awesome** — ícones de redes sociais

## 🎨 Destaques visuais

- **Fundo animado em canvas**: chuva de código estilo Matrix, com colunas de caracteres (katakana, números e símbolos) caindo em ciano e roxo neon
- **Tema neon**: paleta ciano (`#00f3ff`) e roxo (`#9d00ff`) sobre fundo escuro
- **Animações de scroll**: elementos surgem suavemente conforme aparecem na tela, via `IntersectionObserver`
- **Totalmente responsivo**: menu hambúrguer animado (3 barras → X) para mobile, layout adaptado para tablets e celulares
